# .github Repo — Genfeed.ai Org Standards

Centralized GitHub community standards and reusable workflows for all Genfeed repositories.

## Scope

- Pull request template
- Reusable GitHub Actions workflows
- Org-wide workflow defaults

## Rules

- Keep templates generic enough for multi-repo use.
- Favor reusable workflows over copy-paste per repo.
- Changes here can impact many repos; keep updates backward compatible where possible.
